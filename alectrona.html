Overview

Alectrona is an iPad application intended to be used for creating electronic music. This projects focus was to create a design system as well as custom interface themes for user personalization.

Role

Role: Junior UX Designer - Graphic Designer
Product: iPad Application

Responsibilities

Design all buttons, components, and graphic icons. Create themes for interface.

Goal

The objective of this project was to create an application that enables users to create music effortlessly on their iPad device. The project required building a flexible design system suitable for both this project and future adaptations.

As the sole Graphic Designer, I was responsible for creating all icons, buttons, and components, as well as create themes for user's to select, that would transform the appearance of the interface. These themes were essential for fostering user engagement by enabling the app to align with personal identities, encouraging more frequent usage. I specifically aimed to appeal to traditional demographics while also addressing groups often overlooked in the music technology market.


    Challenges

  • Design elements must maintain scalability and flexibility across all implementations
  • Develop themes that connect with users on a personal level
  • Create original custom icons from scratch for all interface buttons
  • Customize buttons specifically for each theme while maintaining functionality

Disclaimer: this project is ongoing.

research

Components

In order to design icons that would be recognizable and familiar to users, I looked at interfaces for pre-existing electonic insturments. An essential part of my research was a book by the name of "PUSH TURN MOVE: Interface design in electronic music" by Kim Bjorn. I analyzed dozens of images of music interfaces and the visual look of the buttons, sliders and knobs so that they could be recreated for this project. For some of the themes, the goal was realism, others minimalism.

design process

Brainstorming

I started with a mobile-first approach for scalable designs across devices. The most important features I needed to include were: a prominent contact button and a hero message.

Initially planning light/dark mode with a toggle. To create a tech-focused aesthetic, I selected fonts that reinforced this vision. The high-contrast combination of neon colors against a black background evoked a programming environment. I ultimately committed to a dark mode-only design as it best served the concept. The BOOK CALL button required prominent homepage placement.

Laying out the pages

With the dark mode esablished as the main design, I began laying out the other pages and creating polished designs for each one.
